Proceso de Elaboración de Chocolate MentaCao

Ingredientes

  • Cacao
  • Azucar
  • Agua
  • Miel
  • Menta
  • Maizena
  • Leche
Paso 01

Preparación del cacao

Se extrae el cacao y se limpia para eliminar impurezas. Luego se deja listo para comenzar el proceso de preparación del chocolate.

Paso 02

Tostado del Cacao

Los granos de cacao se tuestan para resaltar su sabor y aroma característico del chocolate.

Paso 03

Triturado

El cacao tostado se tritura o muele hasta obtener una pasta de cacao que servirá como base del chocolate.

Paso 04

Preparación de la mezcla

Se coloca la pasta de cacao en un recipiente y se agrega azúcar y un poco de agua, mezclando bien para empezar a formar el chocolate.

Paso 05

Calentar la leche

En otro recipiente se hierve la leche para preparar la capa cremosa que llevará sabor a menta.

Paso 06

Agregar la menta

A la leche caliente se le añade la menta, mezclando poco a poco para que el sabor se integre.

Paso 07

Espesar la mezcla

Se mezcla bien y se agrega maizena para espesar la preparación y lograr una textura más cremosa para el relleno.

Paso 08

Añadir miel

Finalmente se agrega miel para darle un toque de dulzura natural y mejorar el sabor del relleno de menta.

Paso 09

Colocación del chocolate

Se derrite el chocolate y se coloca una primera capa en el molde donde se va a formar la barra. Luego se deja enfriar un poco para que empiece a solidificarse.

Paso 10

Agregar la crema de menta

Cuando la primera capa está firme, se agrega la crema de leche con menta en el centro. Después se coloca otra capa de chocolate encima y se vuelve a refrigerar para que tome forma.

Paso 11

Empaque

Cuando la barra está completamente firme, se desmolda y se empaqueta, quedando lista para su distribución o comercialización.
